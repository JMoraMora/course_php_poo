# Relacion con multiplicidad asociativa

Las relaciones asociativas pueden ser bidireccionales como unidireccional. 

| Mult | Descripcion |
| --- | --- |
| 1 | Uno y solo uno |
| 0 .. 1 | Cero o uno |
| N .. M | Desde N hasta M |
| * | Cero a varios |
| 0 .. * | Cero a varios |
| 1 .. * | uno a varios, al menos uno |
